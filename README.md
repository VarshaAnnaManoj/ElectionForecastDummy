# Kerala Assembly Constituencies

A simple static web project that displays all 140 Kerala Legislative Assembly constituencies.

## Files

- `index.html` — main page
- `style.css` — visual styling
- `app.js` — constituency data and search logic

## Usage

1. Open `index.html` in your browser.
2. Use the search box to filter constituencies by name.

## Notes

- This project is a static client-side application.
- The constituency list is based on the current Kerala Assembly constituencies.
